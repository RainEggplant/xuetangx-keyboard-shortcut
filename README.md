# XuetangX Keyboard Shortcut

增加 [学堂在线](https://next.xuetangx.com/) 视频对快捷键的支持。当前只适配了新版（域名以 next 开头）学堂在线。

## 快捷键

| 键       | 效果       |
| -------- | ---------- |
| 空格     | 播放/暂停  |
| 方向左键 | 倒退 10 秒 |
| 方向右键 | 前进 10 秒 |

经试验，无法通过修改新版学堂在线视频的 `video` 元素的 `volume` 和 `playbackRate` 属性改变音量和播放速度（修改后会立刻被还原）。如果你知道解决办法，请告诉我。
